# CLAUDE.md

When working in a repository, scan the root for a `conductor/` directory.

If `conductor/` exists:

1. Read the Conductor project context before changing code.
2. Use the Universal File Resolution Protocol from `conductor-core/context/GEMINI.md` when resolving Conductor artifacts.
3. Run the Conductor status workflow before starting other work.
4. Continue the active track and current phase instead of inventing a new workflow.
5. Preserve Conductor artifacts as normal tracked project files.

Primary lifecycle skills:

- `conductor-setup`
- `conductor-newTrack`
- `conductor-implement`
- `conductor-status`
- `conductor-review`
- `conductor-revert`
